# Переменные Окружения (Environment Variables)

Создайте файл `.env.local` в корне проекта и скопируйте туда содержимое ниже.
Не забудьте подставить свои реальные данные!

```ini
# ------------------------------------------------------------------------------
# 1. БЕЗОПАСНОСТЬ (SECURITY)
# ------------------------------------------------------------------------------

# Email для входа в админ-панель (/admin/login)
ADMIN_EMAIL=admin@lustral.ru

# Пароль администратора
# ВАЖНО: Используйте сложный пароль в продакшене!
ADMIN_PASSWORD=admin123


# ------------------------------------------------------------------------------
# 2. БАЗА ДАННЫХ И ХРАНИЛИЩЕ (SUPABASE)
# ------------------------------------------------------------------------------
# Эти данные нужны для сохранения товаров и картинок в облаке.
# Зарегистрируйтесь на https://supabase.com, создайте проект и найдите ключи в Settings -> API.

# Ссылка на проект (Project URL)
NEXT_PUBLIC_SUPABASE_URL=https://ваша-ссылка.supabase.co

# Публичный ключ (anon / public key)
# Позволяет клиенту читать данные (безопасно, если настроены RLS Policies)
NEXT_PUBLIC_SUPABASE_ANON_KEY=ваш-публичный-ключ


# ------------------------------------------------------------------------------
# 3. НАСТРОЙКИ САЙТА
# ------------------------------------------------------------------------------

# URL вашего сайта
# Используется для формирования ссылок и SEO.
# Для локальной разработки: http://localhost:3000
# Для Vercel: https://ваш-проект.vercel.app
NEXT_PUBLIC_SITE_URL=http://localhost:3000
```
