# Инструкция по деплою и безопасности

## Безопасность
На текущий момент проект имеет базовую защиту:
1. **Админ-панель**: Логин перенесен на Server Actions, что скрывает пароль из клиентского кода браузера.
2. **Переменные окружения**: Пароли не хранятся в коде, а берутся из `process.env`.

### Что нужно сделать в Vercel:
При деплое в настройках проекта (Settings -> Environment Variables) добавьте:
- `ADMIN_EMAIL`: ваш email
- `ADMIN_PASSWORD`: сложный пароль

## Ограничение: Локальное хранилище (LocalStorage)
**ВАЖНО**: Проект сейчас работает в режиме **MVP (Prototype)** без настоящей базы данных.

Все данные (новые товары, изменения в Hero, корзина) хранятся в **LocalStorage вашего браузера**.
Это означает:
1. Если вы добавите товар через админку, **его увидите только вы в этом браузере**.
2. Другие пользователи, зашедшие на сайт с другого устройства, **не увидят ваши изменения**.

### Как исправить для настоящего магазина?
Для полноценной работы необходимо подключить базу данных (PostgreSQL, MongoDB или Supabase).
Это требует доработки проекта:
- Замена `HeroContext` и `ProductContext` на API запросы к базе данных.
- Настройка Prisma ORM или другого клиента БД.

## Инструкция по деплою на Vercel

1. **GitHub**:
   - Код уже загружен в репозиторий: [https://github.com/azabirov/lustral](https://github.com/azabirov/lustral)

2. **Vercel**:
   - Зайдите на [vercel.com](https://vercel.com)
   - Нажмите "Add New..." -> "Project"
   - Найдите репозиторий `lustral` и нажмите **Import**.
   - В разделе **Environment Variables** добавьте:
     - `ADMIN_EMAIL`: `admin@lustral.ru` (или свой)
     - `ADMIN_PASSWORD`: `admin123` (или свой сложный пароль)
   - Нажмите **Deploy**.

Сайт будет доступен по ссылке, которую выдаст Vercel.
